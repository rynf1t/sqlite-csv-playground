# SQLite CSV Playground Pro

Execute SQL queries and analyze CSV data directly in your browser with support for query chaining, visual query building, and real-time autocomplete.

## Live Demo

[Try it here](https://rynf1t.github.io/sqlite-csv-playground/)

## Running Locally

```bash
git clone https://github.com/rynf1t/sqlite-csv-playground.git
cd sqlite-csv-playground
open index.html
```
